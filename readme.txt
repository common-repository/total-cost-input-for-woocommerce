== Description ==

Total Cost Input for WooCommerce plugin allows customers to input the amount of money they want to spend and let the system calculate the resulting quantity.

This plugin is ideal to sell products that can not be counted in portions, like liquids, powders, gas, cryptocurrencies, etc.

With this plugin used, your customer can just enter a sum, like $100 for example, and buy some amount of product you are selling, in grams, litres, bitcoins, etc.

== FREE Features ==

* Turn all products in your WooCommerce store to the `Total Cost Input` type product

== PRO Features ==

The [PRO](https://ethereumico.io/product/woocommerce-input-quantity-total-cost-wordpress-plugin/) version includes all FREE features, plus:

* Turn products in your WooCommerce store to the `Total Cost Input` type product on a product per product base
* Option to show both, the `Total Cost Input` and `Quantity` fields
* Dynamic Quantity and Money fields display and update
* The `Product Quantity Step` setting to allow fractional product quantities

== Installation ==

* Install and activate it as you would any other plugin.
* Make a new WooCommerce product, or just check the `Total Cost Input` checkbox on an existing one.

https://youtu.be/5O4uBHYiRo8
